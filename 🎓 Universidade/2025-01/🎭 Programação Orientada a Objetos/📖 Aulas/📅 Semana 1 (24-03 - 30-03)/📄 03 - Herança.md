# Herança  
**Data:** 09/05/2025  

### Princípios:  
- **Extends**:  
  ```java
  class Animal { void comer() { /* ... */ } }
  class Cachorro extends Animal { void latir() { /* ... */ } }
```
- **Super**:
    
    - `super()` chama o construtor da classe pai.
        

### Cuidados:
- Evitar herança múltipla (Java não suporta).
